I used the following technologies to build:

  • Python
    • FastAPI - To Host Backend Server
    • SQLAlchemy - Mainly connection for remote server To store all the data I'm retrieving
    • Youtube API - To get latest records every 60 seconds
  • Microsoft Azure
    • Container Apps - Host my server on Azure
    • Azure PostgreSQL - Remote database for storing data
  • Docker - For Containerisation of app

To Run the Server Locally, You Need:

  • PostgreSQL Database
  • 3 YouTube API Keys For overcoming problem of Quota

Required Environment Variables:

  • YOUTUBE_API_KEY - Your Youtube API key
  • DB_NAME - Database name
  • DB_USER - Database username
  • DB_PASS - Database password
  • DB_HOST - Database host URL
  • DB_PORT - Database port
  • SSL_CERT_PATH - SSL certificate path for secure connections
